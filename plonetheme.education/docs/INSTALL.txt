plonetheme.education Installation
---------------------------------

To install plonetheme.education using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``plonetheme.education`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        plonetheme.education
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
